# IRIS_FLOWER_DATA_ANALYSIS
This project analyzes the Iris dataset using Python to explore feature relationships and classify flower species. It includes data visualization, exploratory analysis, and machine learning models to predict species accurately and understand key patterns in the dataset.

# 🌸 Iris Dataset Analysis & Classification

## 📌 Overview
This project analyzes the famous Iris dataset to understand patterns among different flower species and build a basic classification model. It demonstrates data analysis, visualization, and machine learning techniques.

## 🎯 Objectives
- Perform exploratory data analysis (EDA)
- Visualize feature relationships
- Understand species distribution
- Build a classification model
- Evaluate model performance

## 🛠️ Tools & Technologies Used
- Python 🐍
- Pandas
- NumPy
- Matplotlib & Seaborn
- Scikit-learn

## 📂 Dataset
- Iris Dataset (built-in / CSV)
- Features:
  - Sepal Length
  - Sepal Width
  - Petal Length
  - Petal Width
- Target:
  - Setosa, Versicolor, Virginica

## 📊 Key Insights
- Setosa is easily separable from other species
- Petal features are most important for classification
- Strong correlation between petal length & width
- Clear clustering among species
